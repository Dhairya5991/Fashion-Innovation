# 🧥 AR Virtual Try-On System

An **Augmented Reality (AR) based Virtual Try-On web application** that lets users try clothes virtually using AI-powered body scanning, avatar generation, and garment fitting.

---

## 🚀 Features

* AR-based virtual clothing try-on
* AI body measurement & avatar generation
* Accurate garment mapping
* E-commerce flow (cart, orders, payment)
* Secure authentication (JWT)
* Dockerized microservices

---

## 🛠 Tech Stack

* **Frontend:** React / React Native, AR.js, Three.js
* **Backend:** Node.js, Express
* **AI Services:** Python, OpenCV, TensorFlow/PyTorch
* **Database:** PostgreSQL / MongoDB
* **DevOps:** Docker, Docker Compose

---

## ⚙️ Run Locally (Docker)

```bash
docker-compose up --build
```

* Frontend: [http://localhost:3000](http://localhost:3000)
* Backend: [http://localhost:5000](http://localhost:5000)

---

## 📦 Run Without Docker

```bash
# Backend
cd backend && npm install && npm start

# Frontend
cd app && npm install && npm start
```
